## Run Locally

Clone the project

```bash
  git clone https://github.com/danielContreras28/unir-martrix-gaussiana.git
```

Go to the project directory

```bash
  cd unir-martrix-gaussiana
```

Install dependencies

```bash
  npm install
```

Start the server

```bash
  npm run start
```
